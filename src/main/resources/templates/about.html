About Todo App

효율적인 할 일 관리 시스템

기술 스택
  • Java 11
  • Spring Boot 2.7.18
  • Spring Security
  • Spring Data JPA
  • Thymeleaf
  • H2 Database
  • Bootstrap 5.1.3
주요 기능
  • Todo 추가/수정/삭제
  • 완료 상태 토글
  • 상태별 필터링
  • 우선순위 설정
  • 마감일 관리
  • 통계 대시보드
